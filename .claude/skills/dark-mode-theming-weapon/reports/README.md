# Reports — dark-mode-theming-weapon

This folder accumulates past audit reports produced by `dark-mode-theming-guardian`. Each report follows the template at `templates/audit-report.template.md`.

**Naming convention:** `{YYYY-MM-DD}-{app-name}-theming-audit.md`

The folder is initially empty. Reports are written here when the user explicitly requests a persisted audit (e.g., "save the audit report").
