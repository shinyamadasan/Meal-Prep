# Reports

This folder collects dated implementation and audit reports produced by `csv-xlsx-import-export-guardian` runs.

## Report format

Each report follows `templates/import-report.md`. Reports are saved here when the Angel is invoked as part of a plan execution (e.g., linked to a feature PRD).

## Naming convention

```
YYYY-MM-DD-<feature-or-ticket>-import-export-report.md
```

Example: `2026-05-20-user-data-importer-import-export-report.md`

The folder starts empty; reports accumulate over time as the Angel is invoked.
