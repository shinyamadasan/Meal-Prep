# ux-ui-weapon

Cursor skill that equips `ux-ui-guardian` to be the long-term owner and enforcer of a deploying product's design system. It encodes the 12-step enforcement procedure (open-the-folder → cite → delta → verify), per-library integration guides for shadcn/ui, Mantine, Lucide-react, and Framer Motion, library-wrapping discipline, and the handoff protocol with `design-system-guardian` for system-level changes.

This skill is generic and versatile. Each repo it ships into supplies its own design-system folder at `library/knowledge-base/<product>-ux-ui/` — including any product-specific non-negotiables, platform-owner directives, or commit-message conventions that should layer on top of the generic procedure.
