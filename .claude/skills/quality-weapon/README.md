# quality-weapon

The Cursor skill that equips the `quality-guardian` Angel to audit completed implementations against their source plan documentation. It encodes the canonical audit procedure (Locate plan → Inventory changes → Cross-reference → Five-axis evaluation → Severity classification → Report) plus the findings-report template, the severity rubric, and a library of worked examples.

This Weapon is the final checkpoint in the `library-guardian` (plan) → implementer → `security-guardian` (security) → `quality-guardian` (QA) loop. It runs after `security-guardian` and before work is marked done. See `SKILL.md` for the entry point and `guides/00-principles.md` for scope, ordering, and cross-Angel handoffs.

Forged as Phase 2 of the Legendary Angel Factory pipeline. The Angel that wields this Weapon lives at `.cursor/agents/quality-guardian.md`.
